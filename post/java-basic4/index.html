9. 运算符

算数运算符 + - * / ++ -- %
加法 减法 乘法 除法 自增 自减 取余
赋值运算符 = += -= *= /= %=
赋值 加上原自身赋值 减去原自身赋值 乘以原自身赋值 除以原自身赋值 取余原自身原自身赋值
关系运算符 > < >= <= == !=
大于 小于 大于等于 小于等于 数值相等 不等于
位运算符 ~ >> << >>>
位运算符不做介绍了！
字符串连接运算符 +
字符串可以相加或者被加生成一个新的字符串
三目运算符 X ? Y : Z
若 x 判断为真，则不去考虑后边的；若 x判断为假，考虑后面的真假。
逻辑运算符 ！ & | ^ && ||
一个false，全false 一个F，整个为F T + F = T/ F + T =T T + T = F/ F + F = F T + T = T，第一个对，第二个不用执行 一个为T，整个为T
  • 除法运算，两个数字都是整数为整数除法，否则表示浮点除法。15 / 2 = 7 | 15.0 / 2 = 7.5
  • 整数被 0 除，会产生一个异常。浮点数被 0 除会得到无穷大或者 NaN 的结果。

  • 了解 Math 类的各种方法
