10. 程序流程控制与作用域

作用域

块（block）即复合语句，是指由若干条 Java 语句组成的语句， 并用一对大括号括起来。 块确定了变量的作用域。 一个块中可以潜逃另一个块中。（双循环就是其中的一个例子）

public static void main(String []args){
  int n;
    
    {
        int m;
    }// m 的作用域仅仅在这里面，而n不仅包裹住 m 的括号内有效，在main方法的括号内同样有效。
}

最小作用域最强原则

  • 局域变量在此方法中，比全局变量在此方法中的作用强。

1. 选择控制

条件判断 - if + else

//----if-----
if(条件判断){
  执行语句;
}
//----if-else-----
if(条件判断){
  执行语句;
}else{
  执行语句;
}
//----if-else+if-else-----
if(条件判断){
  执行语句;
}
else if(条件判断){
  执行语句;
}
else{
  执行语句;
}

注意：

  • 执行语句只有一条语句的时候.可以将if esle 的大括号省略

  • ()内是boolean类型表达式，{}是语句块

    • 比较字符串的内容用equals。比较数值用==，比较地址。
    • 基本数据类型：变量名、变量值在栈中。
    • 引用数据类型：变量名在栈中，变量值在常量池中。

选择（分支）- switch

switch(表达式expr){
    case const1:
        statement1;
        break;

    case const2:
        statement2;
        break;

    case const3:
        statement3;
        break;
            … …
    case constN:
        statementN;
        break;
    default:
        statement_dafault;
        break;
}

注意：

  • 表达式必须是int、byte、char、short、enmu、String类
  • constN必须是常量或者final变量，不能是局部变量
  • 所有的case语句的值不能相同，否则编译会报错
  • default可要可不要
  • break用来执行完一个分支后使程序跳出switch语句块，否则会一直会执行下去。
  • 一定要切记在switch循环中，如果没有break跳出语句，每一个case都要执行一遍，再计算最终结果。

if vs switch

- if可以判断范围，也可以判断一个值; switch只能判断指定的值
- 若只判断指定的值，则使用switch语句，效率快; if判断范围，对数据判断灵活，自身的格式也比较灵活

2. 循环控制

while

while 优先判断条件,再执行循环语句。

     while(循环判断条件){
      循环语句;
     }

do-while

do-while优先执行一遍语句,然后再判断条件,若条件允许继续执行循环语句。

do{
  循环语句；
}while(循环判断条件)//括号后没有分号

for

        for (循环变量初始值设定; 循环条件判断; 改变循环变量的值)｛
                循环语句；
        ｝

注意：

  • 必须有条件判断，否则死循环无法跳出。
  • 循环变量必须能够修改，使得循环能够顺利跳出并结束。
  • 若同时省略表达式1，表表达式3，则相当于while（表达式2）语句
  • 三个表达式均省略 即for(;;)语句，此时相当于while(true)语句
  • 表达式1、表达式3可以是逗号表达式,以使循环变量值在修改时可以对其它变量赋值

3. break和continue

  1. break
  • 一旦遇到break，则跳出某个循环，不再执行循环语句。
  1. continue
  • 一旦遇到continue，则略过此次循环，继续执行后面的循环语句。
