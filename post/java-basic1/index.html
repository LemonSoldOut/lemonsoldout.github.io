Java基础（1）

Java  •   阅读量  0  •  阅读时间: 2.3分
Gridea  2020-12-04

1. 开发java应用的要点

  • 一个源文件中只能有一个public修饰的类，其他类个数不限。
  • 一个源文件有n个类时，编译结果的class文件就有n个。
  • 源文件的名字必须和public修饰的类名相同
  • java语言中单词拼写大小写严格区分。
  • main方法入口，且一个项目只有一个main方法。 程序运行从main方法开始。
  • 每一句执行语句以分号 ; 结束
  • 每一段代码块都由{ }包裹住，作用域的体现就出来了。

2. 标识符

  • 命名规则：由字母、下划线、数字和 $ 组成，不能以数字开头，区分大小写，不能是关键字和保留字（goto、const），长度一般不超过15个字符。
  • 驼峰式命名：
    • 类名：单个单词，首字母大写，多个单词，首字母都大写。
    • 方法名、参数名、变量名：单个单词，首字母小写，多个单词，第一单词首字母小写，其他单词首字母大写。
    • 包名：全部小写。

3. 访问权限修饰符

  • 这些修饰符用于控制程序的其他部分对这段代码的访问级别。

5. 进制

2进制: 0 和 1

作业：了解一哈 G·W·Leibniz

  • 逢2进1位。 0(0)...1(1)...10(2)...11(3)...100(4)...
  • 书写规范：0B11/0b11 = 2 + 1 = 3

8进制

  • 逢8进1位。
  • 书写规范：0o11/0O11 = 8 + 1 = 9
  • 不常用

10进制

  • 不做介绍

16进制

  • 逢16进1位。10之后并非11，为A，B，C，D，E，F。

  • 书写规范：0x11/0X11 = 16 + 1 = 17

进制之间的转换

  • 2进制 <-> 8进制
  • 2进制 <-> 10进制

  • 2进制 <-> 16进制

  • 8进制 <- 10进制 -> 16进制


6. 存储单位

bit

  • bit是二进制数的一位包含的信息或2个选项中特别指定1个的需要信息量称为1 bit，是表示信息的最小单位。
  • 只有两种状态：0和1。
  • 这两个值也可以被解释为逻辑值(true/false、yes/no)、代数符号(+/-)、激活状态(on/off)或任何其他两值属性。

byte

  • 1 byte = 8 bit
  • 计算机最小存储单位
